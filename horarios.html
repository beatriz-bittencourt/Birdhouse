Projeto

Este projeto interdisciplinar (que engloba as matérias: Programação WEB, Programação para Dispositivos Móveis e Sistemas Operacionais II) é um site responsivo usando HTML, CSS e JavaScript, capaz de ser visualizado desde telas de computadores até telas de celulares. Foi feito por Beatriz Bittencourt em conjunto com os alunos Gabriel Corrêa, Renan Carlos e Samuel Marques, que ficaram responsáveis por todo o complemento, como a pesquisa e trabalho com a placa ESP32, peças necessárias, fios e a programação da placa.

As ilustrações desse site foram feitas usando Storyset
